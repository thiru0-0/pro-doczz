# pro-doczz

Pro-Doczz is an AI-powered document generation platform that helps professionals and students create high-quality documents quickly and efficiently. Whether you need to generate resumes, client proposals, company policies, or custom documents, Pro-Doczz provides an intuitive interface to streamline your document creation workflow.

## Features

- **AI-Powered Document Generation**: Create professional documents with AI assistance
- **Multiple Document Types**: Generate resumes, proposals, policies, and custom documents
- **User-Friendly Editor**: Edit and customize your documents with an intuitive interface
- **Dashboard**: Manage all your documents in one place
- **Account Management**: Secure user accounts with personalized settings

## Tech Stack

- **Backend**: Node.js with Express
- **Frontend**: HTML, CSS, JavaScript
- **Architecture**: RESTful API with client-server separation

## Getting Started

1. Navigate to the backend directory and install dependencies:
   ```bash
   cd backend
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Open your browser and navigate to `http://localhost:3001`
